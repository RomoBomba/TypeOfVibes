# Type Of Vibes

Небольшая веб-игра-медитация: уютный лужок на вершине горы, старая скамейка и бескрайний закат. Сядь, включи музыку и просто вайбуй.

🌐 **[Живое демо](https://typeofvibes.onrender.com)**

## ✨ Особенности

- 🎮 Вид от первого лица с управлением WASD
- 🪑 Возможность сесть на скамейку и любоваться видом
- 🌅 Смена фонов (5 HDRI-панорам: закат, горы, ночь и др.)
- 🎵 Встроенный аудиоплеер с плейлистом
- 💻 Работает прямо в браузере

## 🛠 Технологии

- [Three.js](https://threejs.org/) — 3D-графика
- [Vite](https://vitejs.dev/) — сборка и разработка
- JavaScript (ES6+)

## 🚀 Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/type-of-vibes.git

# Перейти в папку
cd type-of-vibes

# Установить зависимости
npm install

# Запустить сервер разработки
npm run dev
```

Открой `http://localhost:5173` в браузере.

## 📦 Сборка для продакшена

```bash
npm run build
```
Готовые файлы будут в папке `dist/`.
